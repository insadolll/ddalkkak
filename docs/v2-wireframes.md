### 구성 요소

| 영역 | 컴포넌트 | 비고 |
|------|----------|------|
| 인사말 | H1 + 월 선택 드롭다운 | 사용자 이름, 현재 월 표시 |
| 통계 카드 4개 | `StatCard` × 4 | 진행 프로젝트, 매출, 매입, 수익률 |
| 최근 프로젝트 | 리스트 (최대 5건) | 프로젝트명 + 상태 뱃지 + 경과 시간 |
| 할 일 / 알림 | 리스트 (최대 5건) | 긴급도별 아이콘, D-day 표시 |

### 데이터 흐름

- 통계: 선택된 월 기준 집계 (프로젝트의 확정 견적서 기반)
- 최근 프로젝트: 내 담당 + 최근 업데이트 순
- 할 일: 미처리 견적 회신, 계산서 요청, 마감 임박 등

```

### 그룹핑 옵션

- **상태별** (기본): 진행중 → 완료 → 보류

- **담당자별**: 담당자 기준 그룹
- **월별**: 시작월 기준 그룹

### 세부 구성

#### 가로 타임라인 (상단)

```
●───────●───────◉───────○───────○───────○
수립    견적수령  견적발송  수주확정  납품    계산서

● 완료 노드: bg-primary, 클릭 시 해당 시점 정보 표시
◉ 현재 노드: bg-primary + ring + pulse
○ 미완료 노드: border-gray-300
연결선: 완료 구간 bg-primary, 미완료 구간 bg-gray-200
```

#### 매입 견적서 풀 (아코디언 내부)

- 매입처에서 받은 견적서 원본 PDF + DB 데이터
- 미니 카드 형태로 나열
- "확정" 버튼 → ★ 뱃지 + 테두리 강조
- [+ 견적 등록]: 파일 업로드 + 핵심 정보 수기 입력 폼

#### 매출 견적서 풀 (아코디언 내부)

- 우리가 만든 견적서 (리비전 포함)
- 미니 카드 + 리비전 번호 표시
- [+ 견적 작성]: 매입 견적서 기반 자동 생성 or 직접 작성
- 각 카드에 [메일발송] [PDF] 액션 버튼

#### 첨부 문서 (아코디언 내부)

```
┌────────────────────────────────────────────┐
│ 📎 COA_성적서_03월.pdf      [COA]  03-15  │  ← 파일명 + 유형 태그 + 날짜
│ 📎 MSDS_원료A.pdf           [MSDS] 03-12  │

│ 📎 기타메모.docx             [기타]  03-16 │
│                                            │
│ [+ 파일 업로드]   드래그 앤 드롭 영역       │
└────────────────────────────────────────────┘
```

#### 계산서 / 결제 일정 (아코디언 내부)

```
┌────────────────────────────────────────────────┐
│  매출 계산서                                    │

│  [계산서 요청 메일 보내기]                       │
│                                                 │

#### 요약 패널

## 화면 전환 흐름 (Flow)

### 기본 업무 흐름

```
대시보드
  └→ 프로젝트 목록 → 프로젝트 상세
                        ├→ [+ 매입 견적 등록] → 견적서 편집
                        ├→ [+ 매출 견적 작성] → 견적서 편집
                        ├→ [매출 견적서 자동 생성] → 마진 모달 → 견적서 편집
                        ├→ [발주서 생성] → 견적서 편집 (문구 자동 변경)
                        ├→ [메일 발송] → 발송 확인 모달
                        ├→ [계산서 요청] → 메일 발송 모달
                        └→ [PDF / Excel] → 다운로드

견적서 목록 → 견적서 상세/편집 → [→ 프로젝트 보기] → 프로젝트 상세

매입매출 → 프로젝트 거래 행 클릭 → 프로젝트 상세
        → [+ 단건 등록] → 등록 모달
        → 리포트 탭 → 차트 + 엑셀 다운로드
```

### 딸깍 자동화 흐름

```
[메일 자동 수집]
  IMAP 폴링 → 새 메일 감지 → 수신 견적서 자동 등록
  → 담당자 알림 → 프로젝트 상세에서 확인 → 품목 수기 입력

[견적서 → 발주서]
  프로젝트 상세 → 확정 견적서 카드 → [발주서 생성] → 문구 자동 변경 → 저장

[계산서 요청]
  프로젝트 상세 → [계산서 요청] → 담당자 메일 발송
  → 담당자가 링크 접속 → 프로젝트 정보 확인 → 계산서 처리

[캘린더 자동 등록]
  견적서 확정 시 → 결제조건 파싱 → 마감일 계산 → 캘린더 이벤트 생성
```
